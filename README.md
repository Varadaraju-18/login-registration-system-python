# Login and Registration System (Python + SQLite)

This is a console-based login system developed using Python and SQLite.

## Features
- User registration
- User login authentication
- Data stored in SQLite database

## Technologies Used
- Python
- SQLite

## How to Run
1. Install Python
2. Open terminal in project folder
3. Run: python main.py

## Purpose
Academic mini project to understand authentication logic and database connectivity.
